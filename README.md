# my-gen-ai-core

A modular Python library for controllable generative AI research and development.

## Goal

`my-gen-ai-core` provides reusable building blocks for future GenAI projects, including thesis experiments, image generation apps, benchmarks, and evaluation tools.

## Current Scope

- Model loading
- Generation pipelines
- Utility functions
- Metrics
- Visualization
